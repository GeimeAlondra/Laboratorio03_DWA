# Catálogo de especies animales

App móvil para Android que muestra un catálogo digital de animales con su información básica.

## Características

- Desplazamiento suave para ver todas las fichas
- Cada animal incluye:
  - Imagen representativa
  - Especie (nombre científico)
  - Sexo (Macho/Hembra)
  - Hábitat

## Tecnologías utilizadas

- Android Studio (Kotlin)
- ScrollView + CardView
- Diseño en XML

## Cómo ejecutar

1. Abrir proyecto en Android Studio
2. Conectar dispositivo o iniciar emulador
3. Ejecutar (Run ▶)
